# fy - 命令行翻译工具

fy 是一个简单的命令行翻译工具，基于百度翻译 API 实现。它允许你通过命令行输入要翻译的单词或短语，并将其翻译成指定的目标语言。

## 功能特性

- 目前只支持英译中
- 快速方便，通过命令行直接输入翻译内容
- 提供准确的翻译结果，基于百度翻译 API 实现

## 安装

使用 npm 全局安装 fy：

```shell
npm install -g fy-meow
```
